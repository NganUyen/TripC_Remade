### Hotel Compare

2. **Check "Best For" categories** - AI analyzes which hotel suits different travel purposes
3. **Read warnings** - Important considerations AI identified

1. **Be specific with interests** - More interests = more personalized plan
2. **Use special requests** - Mention dietary needs, mobility requirements, must-see spots

## ğŸ› Troubleshooting

### AI Compare Not Loading

- Ensure you selected at least 2 hotels
- Check internet connection
- Refresh the page
- If persists, basic comparison still works

### Itinerary Generation Fails

- Verify dates are valid (not in past)
- Check trip duration (1-30 days)
- Ensure all required fields filled
- Try simpler destination name (e.g., "Paris" not "Paris, France, Europe")

### Can't View Itinerary

- Check browser doesn't block sessionStorage
- Note: Currently itineraries stored temporarily
- Save to account to keep permanently (coming soon)

### Chatbot Not Responding

- Check if question is clear
- Try rephrasing
- Use specific commands like "Plan trip to [destination]"
- Avoid overly complex requests
